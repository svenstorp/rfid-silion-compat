# Security Policy

## Supported Versions

The latest released version is supported with security updates.

## Reporting a Vulnerability

Please report security issues privately before disclosing them publicly.

Preferred channel:

- Open a private security advisory on GitHub (if enabled), or
- Contact maintainers through the repository issue tracker and request a private channel:
  https://github.com/svenstorp/rfid-silion-compat/issues

When reporting, include:

- Description of the vulnerability
- Affected versions/commits
- Reproduction steps or proof of concept
- Potential impact

We will acknowledge receipt as quickly as possible and work with you on
responsible disclosure and remediation.
